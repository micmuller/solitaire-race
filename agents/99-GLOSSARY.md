# Glossary

- matchId: identifier for a match/room
- shuffleMode: "shared" or "split"
- seed: deterministic random seed used for shuffle
- snapshot: authoritative state dump used on join/reconnect
- move: player action sent over protocol
- sys_move: server-generated move (e.g., automated actions)
- bot_move: bot-generated move (must be validated like a player move)
